Getting Started

Welcome to the Weather App documentation. This section provides information on how to get started with the app.

Installation

To use the Weather App, simply open the app in your web browser. There is no need for installation or additional setup.

Dependencies

The Weather App utilizes the following libraries:

  • AOS (Animate On Scroll): AOS is used for animating elements on scroll. You can find more about it here.
  • Fetch API: The app uses the Fetch API to make asynchronous requests to the OpenWeatherMap API.

Using the Weather App

Learn how to use the Weather App to check the weather forecast for different cities worldwide.

API Integration

Details on how the Weather App integrates with the OpenWeatherMap API for real-time weather data.

OpenWeatherMap API Key

The Weather App uses the OpenWeatherMap API to fetch weather data. You need to obtain an API key from OpenWeatherMap and set it in the `main.js` file under the variable `apiKey`. You can get an API key by signing up on the OpenWeatherMap API website.

API Endpoints

The following API endpoints are used:

  • Current Weather: `https://api.openweathermap.org/data/2.5/weather`
  • 5-Day Forecast: `https://api.openweathermap.org/data/2.5/forecast`
